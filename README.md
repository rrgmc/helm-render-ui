# helm-template-preview

`helm-template-preview` is a command-line tool to preview rendered Helm templates.

## Install

Get an executable from the [releases](https://github.com/rrgmc/helm-template-preview/releases) page, or if you have a 
Go 1.25 compiler available:

```shell
go install github.com/rrgmc/helm-template-preview@latest
```
